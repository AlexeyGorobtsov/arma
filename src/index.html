Пропустить инструкции

1.Для начала добавьте дверь

Нажмите кнопку "Добавить дверь", а затем проследуйте дальнейшим инструкциям.

2.Внести необходимые настройки

Задайте имя для вашей двери, которое позволит идентифицировать ее срерди остальных.

3.Добавьте ключ для двери

Вы можете выбрать какие ключи смогут открыть данную дверь. Также можно создать индивидуальный ключ, который открывает ТОЛЬКО эту дверь.

4.Организуйте свою мастер-систему

Добавьте требуемое количество дверей и присвойте каждое из них нужные ключи, чтобы разграничить права доступа.
